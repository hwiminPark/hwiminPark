### Selected Projects

**AI Analytics Chatbot**  
Built a data assistant using LangChain and LangGraph that routes questions between SQL databases and internal documents. The RAG pipeline combines semantic search with text-to-SQL, pulling from structured and unstructured sources in the same conversation. Added few-shot prompting for the SQL generation step and a Streamlit interface so non-technical users can ask questions without writing code.

**Crash Severity Model (ICBC)**  
Trained an XGBoost model on 300k+ ICBC crash records and used SHAP to pull out the structural risk factors behind severity: high-risk intersections, parked vehicles, and pedestrian or cyclist exposure. Framed the findings as safety recommendations relevant to urban planning and city decision-making.

**[Airbnb Data ELT Pipeline](https://github.com/hwiminPark/airbnb-dbt-snowflake)**  
ELT pipeline: AWS S3 to Snowflake to dbt Core (Medallion Bronze/Silver/Gold), with incremental models, SCD Type-2, star schema plus OBT, tests, Jinja macros, and docs for BI-ready analytics.

**[Book Recommender ETL Pipeline](https://github.com/hwiminPark/book-etl-pipeline-airflow-databricks)**  
Batch ETL pulling from the NYT Best Sellers API, processed with PySpark on Databricks into semantic embeddings, stored in GCS, orchestrated weekly with Apache Airflow.

**Precision and Personalized PAP Therapy**  
Tested several models (Random Forest, Gradient Boosting, SVM, Logistic Regression) to evaluate PAP therapy outcomes. Built the preprocessing pipeline and turned the statistical findings into visualizations for stakeholders.

**Residential Water Demand Modeling**  
Econometric models plus an interactive Streamlit app for City of Calgary policy simulation. Estimated price elasticity of -0.2 and found demand highly responsive to temperature.
