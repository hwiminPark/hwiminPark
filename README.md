### Selected Projects

**[Airbnb Data ELT Pipeline](https://github.com/hwiminPark/airbnb-dbt-snowflake)**  
Built ELT pipeline: AWS S3 → Snowflake → dbt Core (Medallion Bronze/Silver/Gold), with incremental models, SCD Type-2, star schema + OBT, tests, Jinja macros, and docs for BI-ready analytics.

**[Book Recommender ETL Pipeline](https://github.com/hwiminPark/book-etl-pipeline-airflow-databricks)**  
End-to-end batch ETL: NYT Best Sellers API → PySpark on Databricks → semantic embeddings → GCS, orchestrated with Apache Airflow for weekly workflows.

### Academic Projects

**Crash Severity Model (ICBC)**  
XGBoost + SHAP on 300k+ crash records to identify structural risk drivers (high-risk intersections, parked vehicles, pedestrian/cyclist exposure). Translated model insights into actionable safety recommendations for urban planners and city officials.

**[Precision and Personalized PAP Therapy](https://github.com/hwiminPark/)**  
Analyzed ML workflows and predictive models (Random Forest, Gradient Boosting, SVM, Logistic Regression) on video data to evaluate PAP therapy outcomes. Built preprocessing pipelines and translated statistical findings into visualizations for stakeholders.

**Residential Water Demand Modeling**  
Econometric models + interactive Streamlit app for City of Calgary policy simulation. Estimated price elasticity of -0.2; found demand highly responsive to temperature.

**Bid-Ask Spread Prediction**  
Random Forest (R²=0.94) on 650k+ options records; Streamlit dashboard for real-time liquidity forecasting across AAPL, GME, SPY.
